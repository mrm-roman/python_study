## 2026.4.3
1. 变量（variables）是什么？就是给数据起名字。
2. 变量命名：只能字母开头，由字母+数字+下划线构成。通常用英文起名，空格用下划线代替。
3. 变量赋值：“=右边”赋值给“=左边”
4. 常见数据类型：
- 整数 `int`
- 浮点数 `float`
- 字符串 `str`
- 布尔值 `bool`
5. 按时
