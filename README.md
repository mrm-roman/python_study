## 2026.4.3 -“变量、数据类型、输入输出”
1. 变量（variables）是什么？可以赋给值的标签
2. 变量命名：只能字母开头，由字母+数字+下划线构成。通常用英文起名，原空格用下划线代替。
3. 变量赋值：=
4. 常见数据类型：
    - 整数 `int`
    - 浮点数 `float`
    - 字符串 `str`
    - 布尔值 `bool`
5. 查看数据类型：`type()`
6. 类型转换：
```python
    int("123")
    float("3.14")

```
7. 输出：`print()`
    - 在 Python 中，print() 函数在默认情况下会在末尾自动帮你敲回车。也就是说，当你写下`print("你好")`时，Python 实际上执行的是`print("你好", end="\n")`。
    - end是print函数里的一个特殊参数，它的意思是：“打印完这句话之后，最后要加点什么？”。当你手动写了 end="..." 时，Python 就会听你的，不再偷偷敲回车了。

    ```python
        print("我叫" + name + "，今年" + str(age) + "岁")  # 变量age需要转换成字符串，因为只有字符串才能连接字符串
    ```

## 2026.4.4 -“运算符与条件判断”、“循环：for / while”
1. 算术运算符
    - `+ - * /`
    - `//` 整除
    - `%` 取余
    - `**` 幂运算
2. 比较运算符：单一判断
    - `>  <  >=  <=  ==  !=`
3. 逻辑运算符：将单一判断组合为多条件复杂判断
    - `and`：（且）只有当 and 两边的条件全都是 True 时，最终结果才是 True
    - `or`：（或）只要 or 两边的条件中哪怕有一个是 True，最终结果就是 True
    - `not`：（取反）翻转结果的。把 True 变成 False，把 False 变成 True
4. 条件判断
    - `if:...`
    - `if:...else:...`
    - `if:...elif:...else:...`
    - ps：`elif`可使用多次；`else`可省略；`if...elif...else`结构在一个条件通过执行后将跳过余下的其余条件。
5. for循环：`for 临时变量 in 可迭代对象:`
    - 可迭代对象通常是列表、字符串或者 range() 生成的一组数字
    - 遍历列表：
        ```python
        parts_list = ["齿轮", "轴承", "螺丝", "马达"]
        for part in parts_list:
            print(f"正在检查零件：{part}")
        ```
    - 遍历字符串：
        ```python
        word = "Python"
        for letter in word:
            print(f"提取出字母：{letter}")
        ```
      - 配合 range() 做固定次数的重复：range(n)会生成0,1,2,3...，n-1这n个数字；range（a,b+1）会生成a,a+1,...b-1,b这b+1-a个数字
        ```python
        for i in range(3):
            print(f"正在进行第 {i} 次测试...")
        ```
    - for循环遍历某列表，如果在列表遍历的过程中找到想要的元素就输出yes，如果遍历完整个列表也没找到想要的，就输出no。怎么写？
      - 1.在 Python 里，for 循环后面也可以直接接 else！它的核心逻辑正是你想要的：如果 for 循环是“自然死亡”（正常遍历完所有元素，没有中途被 break 打断），就会执行 else 里面的代码。 只要中途被打断了，else 就绝对不执行。
      - 2.在循环开始前，先立一个标志变量（Flag），默认是没找到（False）。在循环里如果找到了，就把标志改成 True。循环结束后，看这个标志是什么状态。
6. while循环：只要条件为真，就一直执行
    - 当你不知道要循环多少次，只知道“什么情况下该停下”时，就用 while。
      ```python
        while 条件判断:
            # 只要条件是 True，这里的代码就会一直、反复地执行
      ```
7. 循环控制指令
    - `break`：只要代码碰到了break，程序直接跳出循环体，往下执行。
    - `continue`：遇到 continue，当次循环剩下的代码不执行了，立刻回头跳到循环的最顶端，开始下一轮循环。
    - `pass`：占位符。为什么需要它？ 因为 Python 对语法格式要求极度严格，if 或 while 后面的冒号下方必须有缩进的代码块。如果你在搭程序的骨架，还没想好里面写什么逻辑，为了防止 Python 报错（SyntaxError），就用 pass 占个坑。
8. 循环嵌套
    - 尽量不要超过 3 层嵌套，否则时间复杂度会爆炸。
    - 注意内外层变量名不要混淆。
    - 变量重置位置需正确。

## 2026.4.6 -“字符串”
1. 字符串是什么？
    - 本质是字符序列，用单引号或双引号表示
2. 字符串的索引，切片[start:end:step]
    ```python
        s = "hello"
        print(s[0])   # 输出h
        print(s[1:4]) # 输出ell
        print(s[::-1])  # 输出olleh（翻转字符串）
    ```
3. 字符串常见操作
    - 拼接 `+`
    - 重复 `*`
    - 成员判断 `in`
    - 长度统计 `len`
4. 字符串常用方法
    - `upper()`：把字符串里的所有英文字母变成大写
    - `lower()`：把字符串里的所有英文字母变成小写
    - `title()`：把字符串里的每个单词首字母大写
        ```python
            part_id = "PaSsEd-woRD"
            print(part_id.upper())  # 输出: PASSED-WORD
            print(part_id.lower())  # 输出: passed-word
            print(part_id.title())  # 输出: Passed-Word
        ```
    - `strip()`：去掉字符串最左边和最右边的空白字符（包括空格、回车符 \n、制表符 \t）；注意，它不管中间的空格。
        ```python
            raw_data = "   1050度   "
            print(raw_data.strip())   # 输出: 1050度  (干净利落)
        ```
    - `replace()`：把字符串里的“旧词”全部换成“新词”。如果把“新词”写成空字符串 ""，就相当于删除操作。
        ```python
            text = "当前压力：1.5MPa，温度：1.5度"
            print(text.replace("1.5", "2.0"))      # 将所有的 1.5 替换成 2.0，输出: 当前压力：2.0MPa，温度：2.0度
            print("1200℃".replace("℃", ""))     # 删除不需要的单位℃，输出: 1200
        ```
    - `split()`：拆分。它像一把刀，按照你指定的“分隔符”（比如逗号、空格、横线），把一长串文字切成好几块，并装进一个列表 (List) 里。如果什么都不填，默认按所有的空白字符切，此时它会自动忽略首尾的空白，自动把中间连续的多个空格当成一个整体切掉。
    - `join()`：缝合。它是 split() 的反操作。用一个你指定的“胶水（字符串）”，把一个列表里的多个元素重新粘合回一长段文字。
        ```python
            sensor_row = "2026-04-06,Sensor_A,1050,Normal"
            data_list = sensor_row.split(",")   # 按照逗号切开
            print(data_list)    # 输出: ['2026-04-06', 'Sensor_A', '1050', 'Normal']

            sensor_row_new = " & ".join(data_list)    # 用 " & " 作为胶水把它们粘起来
            print(sensor_row_new)     # 输出: 2026-04-06 & Sensor_A & 1050 & Normal
        ```
    - `find()`：在文本里寻找某个特定的词。如果找到了，它会告诉你这个词从第几个字符开始出现（索引位置）；如果没找到，它会返回 -1。
    - `count()`：统计某个字或词在整段文字里出现了几次。
    - `startswith()`：判断字符串是不是以某个特定的词开头的，返回布尔值（True 或 False）。
    - `endswith()`：判断字符串是不是以某个特定的词结尾的。
        ```python
            history = "加工开始...重试...温度异常...重试...加工完成"
            print(history.find("重试"))   # 输出: 7
            print(history.count("重试"))  # 输出: 2
            print(history.startswith("重试"))  # 输出:False
            print(history.endswith("加工完成"))  # 输出:True
        ```
5. 格式化输出
    - `f-string`：见2026.4.3 - 7. 输出：print()中的最后一条
    - `format()` ：使用场景为需要先定义好一个带有坑位的“模板”，等程序运行到后面再去填它。
        ```python
            pi = 3.1415926
            ratio = 0.854
            id_num = 7

            print(f"圆周率约等于：{pi:.2f}")
            # 保留两位小数： {:.2f} (f 代表 float 浮点数)
            print("合格率为：{:.1%}".format(ratio))
            # 转为百分比： {:.1%} (保留 1 位小数的百分比)
            print("零件编号：{:0>3}".format(id_num))
            # 用 0 补齐位数： {:0>3} (总共 3 位，不够的在左侧用 0 补齐)
        ```

## 2026.4.7 -“列表”
1. 列表定义  [ ]
    - 一种数据结构，关联的数据整合在一起
    - 列表可以存多个值（元素）
    - 元素可以修改
2. 列表索引：定位元素 `列表名[位置]`
    - 正向索引：0,1,2,3,...
    - 负向索引：-1,-2,-3,...
3. 列表切片：提取一组连续的元素 `列表名[开始：结束：步长]`
    - [开始：结束]为左闭右开
4. 列表常用方法：
    - `append()`：把新元素放列表末尾       `列表名.append(新元素)`
    - `insert()`：把新元素放在指定位置     `列表名.insert(指定位置 , 新元素)`
    - `extend()`：把另一个列表的元素依次排到当前列表的最后   `列表名.extend(另一个列表)`
    - `remove()`：指定某元素，删除列表中第一个出现的该元素；若列表中找不到该元素，则报错。 `列表名.remove(指定元素)`
    - `pop()`：删掉列表中某位置的元素并可提取赋值；若不指定位置则默认删掉最后一个。`指定位置的元素 = 列表名.pop(指定位置)`
    - `clear()`：清空列表中的元素。`列表名.clear()`
    - `index()`：查找指定元素在列表中第一次出现的位置。`位置 = 列表名.index(指定元素)`
    - `count()`：统计某元素在列表中出现几次。`出现次数 = 列表名.count(指定元素)`
    - `sort()`：把列表元素从小到大，从a到z排序。排序后会直接修改原有列表。`列表名.sort()`  或者从大到小：`列表名.sort(reverse = True)`
    - `reverse()`:把现有列表首尾翻转。排序后会直接修改原有列表。`列表名.sort()`
5. 列表遍历：
    - 元素遍历：for item in list
    - 数遍历：for i in range(len(list))
6. 列表推导式
    - `列表名 = [元素生成逻辑  for 变量 in 原始列表]`
    - 用于快速生成新列表：把创建空列表，写for循环，append（）塞数据这三个步骤压缩成一行代码。
    - 举个例子：
        ```python
            squares = [x * x for x in range(1, 6)]
            #列表squares的元素为1,4,9,16,25
        ```
7. 元组Tuple  (  )
   - 和列表相似，但不可修改
8. 集合Set  {  }
    - 集合中的元素：无序、不重复。常用于列表去重，或者集合运算。
    - `add()`：往集合添加新元素。 `集合名.add(新元素)`
    - `remove()`：删除指定元素。若集合中不存在指定元素，会报错。 `集合名.remove(指定元素)`
    - `discard()`：删除指定元素。若集合中不存在指定元素，不报错。`集合名.discard(指定元素)`
    - 并集：`新集合 = 集合A | 集合B` 或 `新集合 = 集合A.union(集合B)`
    - 交集：`新集合 = 集合A & 集合B` 或 `新集合 = 集合A.intersection(集合B)`
    - 差集: `新集合 = 集合A - 集合B` 或 `新集合 = 集合A.difference(集合B)`
9. 【总结】列表 Vs 元组 Vs集合
    - 列表[] ; 元组(); 集合{}
    - 列表有序，元组有序，集合无序
    - 列表可变，元组不可变，集合可变
    - 列表允许重复，元组允许重复，集合不允许重复

## 2026.4.8 -“字典”、“函数入门”
1. 字典定义  {键:值}
    - 字典是“键值对”数据结构
2. 字典常见操作
    - 访问值 `字典名[键]`——>如果找不到这个键，会报错
    - 新增键值对 `字典名[新键] = 新值`
    - 修改值 `字典名[键] = 新值`
    - 删除键值对 `del 字典名[键]`
3. 字典常用方法
    - `get()`：获取键的对应值，如果找不到返回None `字典名.get(键)——>返回对应的值/None`
    - `keys()`：把字典中所有键单独提取到列表中 `字典名.keys()`
    - `values()`：把字典中所有值单独提取到列表中 `字典名.values()`
    - `items()`：把字典中所有键值对打包成一对对的元组提取，常用语字典遍历 `字典名.items()`
    - `pop()`：删除字典中指定键值对，并返回被删除的值 `字典名.pop(键)——>返回对应被删除键值对的值`
    - `update()`：将新字典内容合并进原字典中，若有新键则增加键值对，若为已存在的键则覆盖原来的值 `原字典名.update(新字典名)`
4. 遍历字典
    ```python
    for k, v in student.items():
        print(k, v)
    ```
5. 字典应用场景:
    - 记录成对的信息
    - 统计字符出现次数
    - 配置项存储
6. 函数定义：`def 函数名(参数)`
    - 为什么要用函数？减少重复代码，让程序更清晰，方便维护和复用
7. 函数的参数
    - 位置参数：严格按照函数定义时从左往右传递参数
    - 默认参数：在函数定义时给参数设一个默认值
    - 关键字参数：在调用函数时，直接把参数名写出来赋值
8. 函数的返回值 return
    - 用法：`return 变量`
    - return后的变量可作用于函数范围之外，没有return的变量只能作用于函数内部
    - 如果返回多个变量：（如return a, b, c）
       - 明确所有的return变量：x, y, z = function()，#x-a，y-b，z-c
       - 只取其中要用的return变量：x, _, _ = function()，#x-a
       - 将return的所有变量打包成元组取出：pos = function()，#pos-(a,b,c)
9. 函数的递归
    - 函数自己调用自己
    ```python
    #计算阶乘
    for k, v in student.items():
    def factorial(n):
        # 基线条件（出口）：如果 n 降到了 1，就知道结果是 1，不能再往下乘了
        if n == 1:
            return 1
    
        # 递归条件（推进）：如果 n 不是 1，就把问题缩小：当前的 n 乘以 (n-1) 的阶乘
        else:
            return n * factorial(n - 1)

    result = factorial(3)
    print(f"3的阶乘是: {result}")
    ```
